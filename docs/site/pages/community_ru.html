Community

Deckhouse is an Open Source project, and everyone is warmly welcome to contribute to it on GitHub.

To make Deckhouse development vigorous and sustainable, its platform is offered in two editions:

Community Edition (CE)

An Open Source and free of charge version providing key functionality of Deckhouse. Its code is available under the Apache 2.0 license.

Enterprise Edition (EE)

A commercial version that boasts additional features (please check details here) and various subscription options. Its code is also open, but it’s not free, and the license is not Open Source.
The development process of both editions is public.

Technical core

Deckhouse follows the upstream version of Kubernetes, building all its features and configurations on top of it.

The basic functionality of Deckhouse is implemented via two building blocks that have been available as Open Source projects since April/May 2019.

Shell-operator

To create Kubernetes operators

Addon-operator

To consistently manage modules based on shell-operator
