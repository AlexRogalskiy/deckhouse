Deckhouse Platform EE

What is it?

Enterprise Edition provides you an easy-to-use way to benefit from self-hosted Deckhouse Platform. Deploy it on any infrastructure and enjoy all features of the platform! While your engineers are responsible for maintaining it, Flant is always happy to help with first-hand support and consultations.

EE source code is open, but it’s neither Open Source nor free to use. If your subscription finishes, you can still use and modify the source code of the last purchased EE version. However, in this case, you will stop getting further updates, can’t create new clusters, and are not allowed to provide Kubernetes-as-a-Service to third parties.
