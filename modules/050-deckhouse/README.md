Модуль deckhouse
==============

Модуль не устанавливает, но настраивает Deckhouse.

Конфигурация
------------

### Что нужно настроить?

Ничего!

### Параметры


* `logLevel` — уровень логирования Deckhouse: `Debug`, `Info` или `Error`. По-умолчанию `Info`.

### Пример конфига

```yaml
deckhouse: |
  logLevel: Debug
```
